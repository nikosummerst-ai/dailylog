# Thunderbit

## What It Does
Web scraping tool for extracting data from websites without code.

## How We Use It
- GTM team scrapes event websites and other sources for prospect and event data
- Lighter, non-engineer-friendly alternative to [[Firecrawl]] and [[Apify]]

## Access
- Dashboard: https://thunderbit.com (Pro account)

## Used By
- GTM — event website scraping and data extraction

## Notes
- Pro account held by GTM team
- Preferred over [[Firecrawl]]/[[Apify]] for non-engineers due to ease of use
- Lighter-weight scraping tool — not suitable for large-scale or complex scraping tasks
