# Stripe

## What It Does
Payment processing platform for online transactions.

## How We Use It
- Finance and Events team use for payment processing
- Direct Booking Summit (DBS) conference ticket payments processed via Stripe
- Credit card billing for event registrations

## Access
- Dashboard: https://dashboard.stripe.com (Triptease account)

## Used By
- Finance — payment processing and reconciliation
- Events team — DBS conference ticket sales and billing

## Notes
- Primary use case is events (DBS conference) payment processing
- Handles credit card billing for Direct Booking Summit registrations
