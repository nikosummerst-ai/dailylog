# Perplexity

## What It Does
AI-powered answer engine — combines LLM reasoning with live web search, returning cited answers. Offers Pro Search, Deep Research, and Spaces for persistent context.

## How We Use It
- Fast, cited research for market, competitor, and hotel-industry questions
- Fact-checking and source discovery where [[ChatGPT]] / [[Claude API]] outputs need verification
- Deep Research runs for longer competitive and prospect intelligence

## Access
- App: https://perplexity.ai (Google SSO available)

## Used By
- GTM — daily research, prospect intelligence, fact-checking
- Niko — first stop for anything that needs current web sources

## Notes
- Strong citations — every claim links back to a source, unlike most chat tools
- Complements [[ChatGPT]] / [[Claude API]] / [[Google Gemini]] rather than replacing them
- Use Spaces for recurring research threads (e.g. a specific hotel group, competitor)
