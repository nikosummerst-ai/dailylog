# Axe DevTools

## What It Does
Automated web accessibility testing tool for WCAG compliance checking.

## How We Use It
- Engineering uses for automated accessibility audits during development
- Accessibility champions run WCAG compliance checks on the Triptease platform

## Access

## Used By
- Engineering — automated accessibility testing in the development workflow
- Accessibility champions — WCAG compliance audits
