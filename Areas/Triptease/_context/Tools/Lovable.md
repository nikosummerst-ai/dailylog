# Lovable

## What It Does
AI app builder — generates full-stack web apps (React + Supabase) from natural language prompts. Live preview, GitHub sync, and one-click deploy.

## How We Use It
- GTM — rapid full-stack prototyping of internal tools
- Building working apps with auth and a database without wiring infrastructure manually
- Prototypes that can graduate to production repos via GitHub export

## Access
- App: https://lovable.dev (Google SSO)

## Used By
- GTM — full-stack prototyping and internal tools

## Notes
- Sits alongside [[Bolt]] and [[Replit]] as an AI coding platform — Lovable leans full-stack (Supabase), [[Bolt]] leans one-shot web apps, [[Replit]] leans hosted IDE
- Outputs can be exported to [[GitHub]] and deployed to [[Vercel]] / [[Railway]] for production
- Good for anything that needs a database/auth out of the box
