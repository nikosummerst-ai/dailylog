# Google Analytics

## What It Does
Website analytics platform (GA4) for tracking user behaviour and traffic.

## How We Use It
- Product and CS use for website traffic analysis
- Hotel clients use for their own site analytics
- Attribution comparison between GA4 and Triptease platform data
- Referenced in client reporting alongside Triptease data

## Access
- Dashboard: https://analytics.google.com

## Used By
- Product — platform and website analytics
- CS — client reporting and attribution analysis
- Hotel clients — their own website analytics properties

## Notes
- GA4 (Google Analytics 4) is the current version
- Attribution comparison with Triptease platform data is a common CS workflow
- Often referenced alongside Triptease booking data in client reports
