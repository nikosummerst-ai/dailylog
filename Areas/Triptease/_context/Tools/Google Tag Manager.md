# Google Tag Manager

## What It Does
Client-side tag management system for deploying and managing tracking scripts on websites.

## How We Use It
- PIMs and integrations team use for deploying Triptease scripts on hotel booking engines
- Hotel booking engines fire Triptease scripts via GTM containers
- Required for consent management compliance

## Access
- Dashboard: https://tagmanager.google.com

## Used By
- PIMs (Partner Integration Managers) — script deployment on hotel booking engines
- Integrations team — Triptease tag deployment and consent management

## Notes
- The primary mechanism for hotels to fire Triptease tracking scripts
- Required for cookie consent and consent management compliance
- Integrations team manages GTM containers for client onboarding
