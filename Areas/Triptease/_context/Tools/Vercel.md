# Vercel

## What It Does
Deployment platform for web applications with instant global CDN.

## How We Use It
- Sales team deploys free-plan tools for internal use
- Engineering deploys some internal tools and prototypes

## Access
- Dashboard: https://vercel.com

## Used By
- Sales — lightweight internal tools (e.g., orgmapper.vercel.app)
- Engineering — internal tooling and prototypes

## Notes
- orgmapper.vercel.app is a known deployed tool example
- Free plan used for non-production/internal tooling; differs from main engineering deployments on [[Google Cloud Platform]]
