# Google Stitch

## What It Does
Google Labs' AI design tool — generates UI designs and front-end code from natural language prompts or reference images. Outputs editable HTML/CSS and exports to [[Figma]].

## How We Use It
- Rapid UI mockups for internal tools and automation dashboards
- Converting rough ideas into design-system-aligned screens before handing off to [[Figma]] or [[Bolt]] / [[Replit]] for build

## Access
- App: https://stitch.withgoogle.com (Google SSO)

## Used By
- GTM — UI prototyping for internal tools
- Niko — design asset creation for automation front-ends

## Notes
- Sits between pure design tools ([[Figma]], [[Canva]]) and AI app builders ([[Bolt]], [[Replit]])
- Useful when you want design output to hand to a developer or AI coding tool, not a finished app
- Still labelled experimental (Google Labs) — availability/features may change
