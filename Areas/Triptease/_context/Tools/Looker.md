# Looker

## What It Does
Internal analytics and BI dashboards for platform metrics and client performance data.

## How We Use It
- Product, CS, Engineering, and Sales access dashboards for data-driven decisions
- Platform metrics, booking reports, and client performance data
- Built on top of [[Google BigQuery]]

## Access
- Dashboard: https://triptease.looker.com

## Used By
- Product — platform metrics and feature performance tracking
- CS — client performance and health reporting
- Engineering — technical metrics and monitoring dashboards
- Sales — pipeline and commercial data analysis

## Notes
- Built on top of [[Google BigQuery]] as the underlying data layer
- Internal tool only; client-facing reports use [[Looker Studio]] instead
