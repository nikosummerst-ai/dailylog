# Manus

## What It Does
Autonomous AI agent platform — runs multi-step tasks in a cloud sandbox (browsing, file creation, data gathering, report writing) without step-by-step prompting. Delegate a goal; it plans and executes end-to-end.

## How We Use It
- GTM — long-running research, data gathering, and multi-step automations that would otherwise need a human in the loop
- Competitive intelligence and hotel/market research tasks
- One-shot report generation from a high-level brief

## Access
- App: https://manus.im

## Used By
- GTM — autonomous research and task execution
- Niko — offloading multi-step research/automation work

## Notes
- Runs asynchronously in the cloud — start a task, come back later for the output
- Good fit for tasks that are too involved for [[ChatGPT]] or [[Claude API]] one-shot but too small for a custom automation
- Complements [[Gumloop]] / [[Zapier]] — use Manus for open-ended agent work, those tools for deterministic workflows
