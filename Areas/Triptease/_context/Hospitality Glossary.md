# Hospitality Glossary

Industry terms, abbreviations, and concepts used across the hotel/travel sector. For Triptease-specific slang, see [[Glossary - Triptease]].

---

## Distribution & Booking

| Term | Meaning |
|---|---|
| **OTA** | Online Travel Agency — Booking.com, Expedia, Hotels.com, Agoda. Triptease's primary competitor for guest acquisition |
| **Direct Booking** | Booking made on the hotel's own website, not via an OTA |
| **BE** | Booking Engine — software that powers a hotel's direct booking flow (Sabre SynXis, Amadeus iHotelier, D-Edge, SiteMinder, etc.) |
| **CRS** | Central Reservation System — central inventory + rate management across channels |
| **PMS** | Property Management System — front-desk / on-property operations (Mews, Cloudbeds, Opera, etc.) |
| **GDS** | Global Distribution System — Amadeus, Sabre, Travelport. Used by travel agents |
| **Channel Manager** | Distributes inventory/rates across OTAs, GDS, direct |
| **Disparity / Undercut** | When an OTA shows a lower rate than the hotel direct rate |
| **Parity** | Rate parity — having the same (or better) rate on direct vs. OTAs |
| **Rate Plan** | A specific bookable rate (e.g. flexible, non-refundable, member rate) |
| **Promo Code** | Discount code applied at checkout — used by Triptease for [[Glossary - Triptease#Product and Platform\|Price Match]] |

See [[Product Reference#5. Integrations]] for the booking engines Triptease integrates with.

## Search & Advertising

| Term | Meaning |
|---|---|
| **Metasearch** | Search engines that compare hotel rates across OTAs and direct (Google Hotel Ads, TripAdvisor, Trivago, Kayak) |
| **GHA** | Google Hotel Ads |
| **TPA / Property Ads** | Google Travel Property Ads |
| **TASP** | TripAdvisor Sponsored Placements |
| **PMax** | Google Performance Max — automated Google ads. Triptease deliberately doesn't use PMax for Travel Goals (lower ROAS than its own algorithm) |
| **Paid Search** | Google/Bing search ads (brand, non-brand, competitor) |
| **Retargeting** | Ads shown to users who already visited the hotel site (Facebook, Instagram, Google Display Network) |
| **Free Booking Links** | Free (non-paid) hotel direct links on Google |
| **CPC** | Cost Per Click |
| **CPM** | Cost Per Mille (1000 impressions) |
| **ROAS** | Return on Ad Spend |
| **Dynamic Ads** | Ads populated with the user's specific dates/rates |
| **Lookalike Audience** | Facebook/Google audience built to resemble a hotel's best past bookers |

## Revenue & Operations

| Term | Meaning |
|---|---|
| **RMS** | Revenue Management System — pricing & forecasting (Flyr/Pace, Duetto, BEONx, Ideas, Outrigger) |
| **ADR** | Average Daily Rate |
| **RevPAR** | Revenue Per Available Room |
| **Occupancy** | % of rooms occupied |
| **Lead Time** | Days between search/booking and check-in |
| **Need Date** | A date with low forecasted occupancy needing a sales push |
| **Length of Stay (LOS)** | Number of nights |
| **Party Composition** | Solo / couple / family / group |

---

## Tracking & Attribution

| Term | Meaning |
|---|---|
| **GTM** | [[Tools/Google Tag Manager\|Google Tag Manager]] — script container on hotel sites |
| **GA4** | Google Analytics 4 |
| **Last-click attribution** | Credit goes to the final touchpoint |
| **30-day attribution** | Credit window of 30 days |
| **Booking Journey Attribution** | Triptease's model — tracks every touchpoint and gives nominal cost per booking. Avg ~71 touchpoints per booking |
| **CVR** | Conversion Rate |
| **Anonymous ID** | Session-based ID Triptease assigns to non-cookie-consenting visitors (~20% in GDPR markets), persistent for 30 days |

See [[Product Reference#2.2 Tracking and Attribution]] for the full Triptease setup.

## On-Site Personalization

| Term | Meaning |
|---|---|
| **Messages** | Triptease's on-site nudge / message product. See [[Glossary - Triptease]] for M1/M2 |
| **Price Check** | Shows direct vs. OTA prices on the booking engine |
| **Email Capture** | Lead-gen widget exchanging email for a discount |
| **Nudge** | Urgency / social-proof on-site message |
| **A/B Test** | Compare two message variants to find the winner |

## Commercial Models

| Term | Meaning |
|---|---|
| **Commission** | % of booking value (typical: 7-10% for Triptease, 15-20% for Booking.com) |
| **CPC Model** | Hotel pays per click |
| **SaaS / Subscription** | Flat monthly/annual fee |
| **Cost of Sale** | Total commercial cost as % of booking value |
| **Pay Per Stay (PPS / PPX)** | Triptease commission model |
| **Booking Reconciliation** | Reviewing/cancelling bookings before paying commission |

## Customer Segments

| Term | Meaning |
|---|---|
| **SMB** | Small/Mid Business hotels (Triptease has a dedicated SMB team — currently 134% of budget, see [[OKRs and Strategic Priorities]]) |
| **Mid-Market** | Triptease's primary ICP — LTV:CAC > 10x |
| **Enterprise** | Large chains (Marriott, IHG, Hilton). Triptease has stepped back from this segment |
| **Independent** | Single property |
| **Hotel Group / Collection** | Multi-property brand |
| **Brand / Chain** | Major chain (Hilton, Marriott, Accor) |
| **Hostel / Alternative Accom** | E.g. HI USA, GCP Hospitality / Haka House |
